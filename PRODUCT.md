# Product Context

## Purpose

This project is an interactive educational web page for Chapter 6: industrialization, modernization, and international economic integration of Vietnam.

## Audience

Students and learners reviewing political economy material. The page should feel clear, credible, memorable, and easier to study than a plain document.

## Register

Long-form educational content. Design is part of the learning experience: visual pacing, imagery, motion, and scannable structure should help readers stay oriented.

## Current Product Shape

- Static single-page website in `index.html`.
- Runtime support is provided by `support.js`.
- Content is organized into sections for industrial revolutions, roles, industrialization models, Vietnam's modernization path, international integration, impacts, solutions, conclusion, and an embedded board game guide.

## Design Direction

- Use strong illustrative imagery tied to the subject matter instead of decorative placeholders.
- Keep the existing warm academic palette, but add more contrast, rhythm, and visual hooks.
- Prefer editorial educational storytelling over generic marketing cards.
- Maintain responsive readability for classroom/projector and mobile study contexts.
